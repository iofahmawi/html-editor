## 📝 محرر HTML

أداة ويب متكاملة لكتابة وتحرير كود HTML مباشرة في المتصفح مع معاينة فورية للنتيجة، مصممة لتكون سريعة وسهلة الاستخدام دون الحاجة لأي تثبيت.

### 🔗 تجربة الأداة
* [🚀 معاينة حية للأداة](https://iofahmawi.github.io/html-editor/)

### ✨ المميزات

* **محرر متقدم**: مبني على مكتبة CodeMirror مع تمييز تلوين الكود (Syntax Highlighting) لـ HTML، CSS، وJavaScript.
* **معاينة فورية**: تشغيل الكود ومشاهدة النتيجة مباشرة داخل الأداة.
* **تنسيق الكود**: تنسيق الكود تلقائياً بضغطة واحدة باستخدام مكتبة Prettier.
* **بحث واستبدال**: أداة بحث متكاملة مع إمكانية الاستبدال الفردي أو الاستبدال الكلي.
* **حفظ محلي**: حفظ العمل تلقائياً في المتصفح دون الحاجة لرفع أي ملفات.
* **دعم الوضع الليلي والنهاري**: تبديل سريع بين الوضعين لراحة العين.
* **تطبيق ويب تقدمي (PWA)**: يمكن تثبيت المحرر على الجهاز والاستخدام بدون إنترنت.
* **واجهة عربية كاملة**: تصميم نظيف ومتجاوب مع جميع أحجام الشاشات.

### 🛠 التقنيات المستخدمة

* **HTML5 & CSS3 & JavaScript (Vanilla)**: بدون أي أطر عمل خارجية.
* **CodeMirror**: لمحرر الكود مع تمييز الألوان ودعم البحث.
* **Prettier**: لتنسيق كود HTML تلقائياً.
* **Service Worker**: لدعم PWA والعمل بدون إنترنت.

### 🚀 كيفية الاستخدام

1. **اكتب الكود**: أدخل كود HTML في المحرر.
2. **معاينة**: اضغط "تشغيل" لمشاهدة النتيجة فوراً.
3. **تنسيق**: اضغط "تنسيق" لترتيب الكود تلقائياً.
4. **بحث**: استخدم "بحث" للبحث عن نص أو استبداله داخل الكود.
5. **حفظ**: اضغط "حفظ" لحفظ العمل محلياً في المتصفح.

### 📂 ملفات المشروع

* `index.html`: الملف الأساسي الذي يحتوي على الواجهة والمنطق البرمجي.
* `manifest.json`: ملف تعريف التطبيق للتثبيت على الأجهزة.
* `sw.js`: ملف Service Worker للتشغيل أوفلاين.
